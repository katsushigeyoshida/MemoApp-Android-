## 超シンプルメモ帳(MemoApp)
### 画面
[新規][前][次][削除][他]  
[タイトル(作成開始日付時間)]  
[テキスト本文            ]  

### 機能
作成開始の日付時間をタイトルにしてテキストでメモの内容を記述  
メモは日付順にソートされ、[前]ボタンと[次]ボタンで切り替える  
オプションとして[他]ボタンから文字サイズの選択と全データ削除がある
